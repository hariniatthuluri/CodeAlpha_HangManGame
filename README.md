# 🎮 Hangman Game in Python

A clean and interactive text-based Hangman Game built using Python.

## 🚀 Features
- Random word selection
- Hint categories
- Input validation
- Duplicate guess detection
- ASCII Hangman graphics
- Replay option

## 🛠 Technologies Used
- Python
- Random Module
- Loops & Conditions
- Lists & Strings

## ▶ How to Run

1. Install Python
2. Save the file as `hangman.py`
3. Run:

```bash
python hangman.py
